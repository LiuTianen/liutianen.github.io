jmeter-json断言

发表于 2020-09-18 分类于 工具使用

记录jmeter使用过程中的断言使用情况

Json断言

当相应结果为Json时，使用Json断言，会更加方便和快捷。

添加方式，右键快捷菜单 – 断言 –Json断言。界面如下：

Assert JSON Path exists: #元素匹配

Additionally assert value # 等于特定值

Match as regular expression # 符合正则表达式

Expected Value： # 特定值

Expect null #预期空值

Invert assertion(will fail if abovew conditions met) #满足上面的条件则fail

判断方式：
  1. ​ 如果响应结果不是json格式的，fail
  2. ​ 如果json path找不到元素，fail
  3. ​ 如果json path找到元素，没有设置条件，pass
  4. ​ 如果json path找到元素了，不符合条件，fail
  5. ​ 如果json path找到元素，且符合条件，pass
  6. ​ 如果json path返回的是一个数组，迭代判断是否有元素符合条件，有则pass，无则fail
